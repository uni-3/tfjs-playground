# tfjs playground

## npm commands

- build
build for production under `./build`.
```bash
npm run build
```

- start
transcript js and scss.

```bash
npm run start
```

run on `localhost:3000`

- server
start server on `localhost:8080`.
host `./build` file.

```bash
npm run server
```
